第一次搭博客避坑教程

相信大家会偶尔脑袋一热想搭个博客，但总是会出很多Bug于是我就把我踩过的坑及如何处理写了出来。

1.安装Node.js与Git

首先需下载Node.Js和Git，如何下载请自行百度
http://www.baidu.com/

下载完打开CMD，输入:

node -v

还有:

 npm -v

确保无误后进行第二步。

2.搭建Hexo

继续输入：

    npm install hexo-cli -g

以安装Hexo，接下来输入:

    npm install hexo-server

安装HexoServer,接下来进入第三步。

3.运行Hexo

用这条几条命令初始化Hexo:

hexo init 博客文件夹的名字（文件夹必须为空）
      cd 博客文件夹的名字
      npm insttal
 如无报错，那么现在运行
 
     hexo g
      hexo s
  浏览再进入localhost:4000，如果有Hexo的话那么祝贺你！
 ### 4. 绑定Github Pages
 如何创建Github账号与创建Github、 Pages此处不再赘述，此处主要讲绑定Github Pages的方法。
 首先打开_config.yml，找到

  deploy:
    type:''

改为:

   deploy:
     type: git

repo: git@github.com:你的Github名字/你的Github名字.github.io
branch: master
  运行:

    hexo d

如报错那试试:
 
  运行npm install hexo-deployer-git
  把branch: master改为branch: main
  接下来等待Hexo自己Copy到Github上，等待即可。

### 查看效果
  进入你的Github Pages地址以查看效果
， 不知道你的Github Pages地址？那就试试:

http://你的Github名称.github.io
 如果看到了Hexo，那么恭喜你，这个幸运儿！
